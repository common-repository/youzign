== Description ==

Youzign for WordPress is a WordPress plugin that lets you import any of your [Youzign](https://youzign.com) designs in your WordPress posts and pages.

= Features include: =

* access all your Youzign designs from your Wordpress dashboard
* insert your designs in posts, pages and add to your media library
* all edits made on Youzign are reflected in your Wordpress site, instantly

== Installation ==

1. Install Youzign plugin either via the WordPress.org plugin directory, or by uploading the files to your server
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Youzign settings to authenticate to your Youzign account.
